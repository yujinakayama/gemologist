# Gemologist

**Gemologist** is a library for rewriting your Gemfile and gemspec.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'gemologist'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gemologist
